Step 1: Upload Files

In order to solve the Vehicle Routing Problem (VRP), two CSV files needs to be uploaded:

  • Customers Data: Contains information about customer locations, demands, and time windows (if applicable).
  • Vehicles Data: Contains information about the vehicles in the fleet, including vehicle capacities.

Make sure your files are in CSV format and match the required structure (Please refer to the relevant section). If you're unsure about the format, you can download the sample customers file and sample vehicles file to use as a template.
